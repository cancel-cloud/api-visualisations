# AGENTS

## Common Workflows
- Install deps: `pnpm install`
- Run dev server: `pnpm dev` (http://localhost:3000)
- Production build: `pnpm build`
- Start prod server: `pnpm start`

## Validation
- Lint: `pnpm lint`
- Typecheck: `pnpm typecheck`

## Tests
- Unit/integration: `pnpm test`
- Watch mode: `pnpm test:watch`
- E2E: `pnpm test:e2e`
